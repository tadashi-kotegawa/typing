## Key Features

### Ranking Button (ðŸ‘‘)
- **Position**: Fixed top-left (10px from top, 10px from left)
- **Visibility**: Always visible on menu and playing screens
- **Action**: Opens ranking modal
- **Style**: 
  - Background: slate-800 with transparency
  - Hover: Changes to yellow-400
  - Icon: Crown from lucide-react

### Ranking Modal
- **Data**: Top 50 scores from Firestore
- **Sorting**: Descending by score
- **Display**: 
  - Rank number (special styling for top 3)
  - Username (default: "Anonymous")
  - Score (formatted with commas)
  - Date (Japanese format)
- **Loading**: Shows loading animation while fetching
- **Empty State**: Encourages first submission

### Username Input Modal
- **Trigger**: Click "ãƒ©ãƒ³ã‚­ãƒ³ã‚°ç™»éŒ²" on result screen
- **Input**: Text field (max 20 characters)
- **Default**: "Anonymous" if empty
- **Actions**: 
  - Skip: Closes without submitting
  - Submit: Saves to Firebase and shows ranking
- **Feedback**: Loading spinner during submission

### Ranking Data Structure
```javascript
{
  username: string,    // Max 20 characters
  score: number,       // Integer points
  timestamp: Timestamp // Firebase server timestamp
}
```

### Firebase Collections
- **Collection Name**: `rankings`
- **Query**: `orderBy('score', 'desc').limit(50)`
- **Index**: Automatically created on `score` field

## Styling Details

### Color Scheme
- **Primary**: Cyan-400 (default mode), Indigo-400 (legal mode)
- **Accent**: Yellow-400 (ranking, combos)
- **Rankings**:
  - 1st Place: Gold (yellow-500)
  - 2nd Place: Silver (gray-400)
  - 3rd Place: Bronze (orange-600)
  - Others: Slate-600

### Typography
- **Scores**: font-mono font-bold
- **Usernames**: font-bold
- **Headings**: text-2xl to text-6xl font-bold

### Animations
- **Modal Entry**: fade-in with backdrop-blur
- **Loading**: pulse animation
- **Hover**: color transitions (duration-200)

### Responsive Design
- **Mobile**: Full-width modals with padding
- **Desktop**: Max-width constraints (max-w-2xl, max-w-3xl)
- **Scroll**: Overflow-y-auto for long rankings
